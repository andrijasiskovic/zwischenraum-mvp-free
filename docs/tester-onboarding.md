# Testkunden-Onboarding

Dieses Dokument kann als interne Anleitung fuer den Betreiber genutzt werden.

## Coach einladen

1. Als Betreiber in Zwischenraum einloggen.
2. Zu `Einladungen` gehen.
3. E-Mail des Testcoaches eingeben.
4. Rolle `Coach` waehlen.
5. `Code erstellen` klicken.
6. Versandoption waehlen oder Text kopieren.
7. Coach darauf hinweisen:
   - Link oeffnen
   - Code aus E-Mail eingeben
   - Vorname und Nachname eingeben
   - Passwort setzen
   - Branche waehlen
   - danach normal einloggen

## Nachricht an Testcoach

Vorschlag:

```text
Hallo,

ich teste aktuell eine neue Web-App fuer strukturierte Begleitung zwischen Terminen.

Du kannst dir damit einen eigenen Testbereich erstellen, Clients einladen, Aufgaben vergeben und Reflexionen einsehen.

Bitte nutze den Einladungslink und den Code aus dieser Mail. Beim Registrieren waehle bitte deine Branche aus.

Wichtig: Bitte fuer den Test keine sensiblen echten Gesundheits- oder Therapiedaten verwenden.
```

## Coach-Testauftrag

Der Testcoach soll:

1. eigenen Zugang erstellen
2. Branche auswaehlen
3. Branding kurz pruefen
4. mindestens einen Client einladen
5. mindestens eine Aufgabe erstellen
6. Client die Aufgabe abschliessen lassen
7. Reflexion im Dashboard oder Client-Profil suchen
8. Reminder Center kurz pruefen

## Client-Testauftrag

Der Client soll:

1. Einladungslink oeffnen
2. Code eingeben
3. Account erstellen
4. einloggen
5. Aufgabe ansehen
6. Aufgabe abschliessen
7. Reflexion und Status ausfuellen

## Feedback einsammeln

Nach 7 bis 14 Tagen:

- 15 Minuten Gespraech mit Coach
- Screenshots von unklaren Stellen sammeln
- konkrete Verbesserungsvorschlaege notieren
- Zahlungsbereitschaft abfragen
