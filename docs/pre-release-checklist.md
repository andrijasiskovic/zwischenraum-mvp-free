# Pre-Release Checklist

Vor einer Veröffentlichung außerhalb enger MVP-Tests prüfen:

- Branchen-Testumschaltung im Branding sperren oder nur Owner/Admins erlauben.
- E-Mail-Bestätigung für produktive Nutzer aktivieren.
- Einladungssystem auf echten Mailversand umstellen.
- Datenschutztexte, Impressum und Löschkonzept ergänzen.
- Keine sensiblen Gesundheits-/Therapiedaten ohne passende Datenschutzprüfung verwenden.
- Online-URL in Einladungslinks testen.
- Coach-Onboarding mit eigenem Workspace testen.
- Client-Onboarding getrennt vom Coach-Onboarding testen.
- Rollenlogik mit mindestens zwei verschiedenen Coaches pruefen.
